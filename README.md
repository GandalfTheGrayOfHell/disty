# Disty

### Features
- Push/Pull functionality
- No versioning
- Project wise config
- No Dependencies

### Maybe
- HTTPS transfer

### Commands
- `init` - Initialize Disty
- `remote` - Set the location of remote server
- `config` - Configure Username and Password
- `add` - Adds file to track
- `status` - Displays the status of project

- `clone`
- `pull`
- `push`
- `serve`
